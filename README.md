## Description

Spotify clone made with Next.js and Supabase based on the tutorial of [Code With Antonio](https://youtu.be/2aeMRB8LL4o?si=a5Q2jpimiWWKAW2a).

## Tecnologies

- TypeScript
- React
- Next.js
- Supabase

## Requirements

- Node.js ^20.13.1
- Yarn

## Installation

1. Install dependencies:

```bash
$ yarn install
```

2. Run application:

```bash
# development watch mode
$ yarn dev

# production mode
$ yarn build
$ yarn start
```
